# Codex Authoring Example

This example is a fuller docs site that shows how a developer can use Codex to create and maintain an Anydocs project.

Use it when you want realistic information architecture and authoring workflow docs rather than the smallest possible sample.

## What This Example Covers

- Codex-oriented onboarding pages
- longer navigation and page sets
- day-to-day authoring workflow guidance
- MCP-assisted writing context

## Quick Start

```bash
node --experimental-strip-types packages/cli/src/index.ts build examples/codex-authoring-docs
node --experimental-strip-types packages/cli/src/index.ts preview examples/codex-authoring-docs
```
